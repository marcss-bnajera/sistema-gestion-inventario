# sistema-gestion-inventario

## 🚀 Cómo empezar

Este proyecto utiliza **submódulos de Git**. Para asegurarte de que todo el código y sus dependencias se descarguen correctamente, sigue las instrucciones a continuación.

### Clonar el repositorio

Ejecuta el siguiente comando en tu terminal:

```bash
git clone --recurse-submodules [https://github.com/marcss-bnajera/sistema-gestion-inventario.git](https://github.com/marcss-bnajera/sistema-gestion-inventario.git)
```

con esto ya tendras el proyecto con los submódulos clonados
